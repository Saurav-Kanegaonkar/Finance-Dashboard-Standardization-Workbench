# Data Sources

Synthetic source-style data for Power BI finance reporting, dashboard standardization, and SAP Analytics Cloud-ready stakeholder requirements.

- `entities.csv`: 36 reporting metric records.
- `daily_metrics.csv`: 5,040 reporting metric-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
